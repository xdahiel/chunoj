# Chun OJ API 管理文档

## 登录

* 请求接口: login

* 请求方法: post

* 请求参数

  | 参数名   | 含义     | 备注             |
  | -------- | -------- | ---------------- |
  | username | 用户名   | 必填项，长度6-20 |
  | password | 用户密码 | 必填项，长度6-20 |

* 返回参数

  

  | 参数名 | 含义   | 备注 |
  | ------ | ------ | ---- |
  | code   | 状态码 |      |
  | msg    | 信息   |      |
  | data   | 数据   |      |

* 响应数据

  ```json
  {
      code: "200",
      data: "",
      msg: "success"
  }
  ```
